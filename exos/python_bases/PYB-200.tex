Dans les cas où c'est possible, affecter les valeurs aux variables correspondantes à l'aide de l'interpréteur interactif d'IDLE. 
On notera \pyv{var} $\leftarrow$ \pyv{a} pour dire que l'on affecte la valeur \pyv{a} à la variable \pyv{var}.

  \begin{enumerate}[label=\emph{\alph*)}]
    \item \pyv{ArthurDent} $\leftarrow$ \pyv{42}
    \item \pyv{4} $\leftarrow$ \pyv{0.}
    \item \pyv{L} $\leftarrow$ \pyv{[]}
    \item \pyv{list} $\leftarrow$ \pyv{[1,2,3]}
    \item \pyv{int} $\leftarrow$ \pyv{5}
    \item \pyv{s} $\leftarrow$ \pyv{""}
    \item \pyv{True} $\leftarrow$ \pyv{1}
    \item \pyv{ok} $\leftarrow$ \pyv{ok}
    \item \pyv{x} $\leftarrow$ \pyv{"x"}

    \item \pyv{or} $\leftarrow$ \pyv{"xor"}
  \end{enumerate}
